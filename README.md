# Rahul Shirode Portfolio

Modern developer portfolio built using

- Next.js
- React
- TypeScript
- Bootstrap
- Git
- GitHub
- AWS (Deployment)
- GitHub Actions (CI/CD)

## Features

- Responsive portfolio
- Dark Mode
- Resume Download
- Contact Form
- Modern UI

## Local Setup

npm install

npm run dev

Visit:

http://localhost:3000

## Deployment

AWS

CI/CD using GitHub Actions
